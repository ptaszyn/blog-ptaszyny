Poruszanie się po katalogach

```bash
$ ls # listuje zawartość folderu w którym się znajdujesz
$ cd <podfolder> # wejście do podfolderu
$ cd /<folder>/<podfolder> # przejście do innego podfolderu
$ mkdir # utworzenie nowego folderu
$ touch <nazwa>.<rozszerzenie> # utworzenie nowego pliku
$ rm <nazwa>.<rozszerzenie> # usunięcie pliku lub pustego folderu
$ rm -rf <folder> # usunięcie folderu wraz z całą jego zawartością
```
